### Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up environment variables:
   - Create a `.env` file
   - Add Supabase credentials

4. Start the development server:
   ```bash
   npm run dev
   ```
